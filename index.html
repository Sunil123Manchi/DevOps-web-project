DevOps Lifecycle

The DevOps lifecycle includes the following stages: Plan, Code, Build, Test, Release, Deploy, Operate, Monitor.

DevOps Advantages and Disadvantages

Advantages:

  • Faster delivery of features
  • More stable operating environments
  • Improved communication and collaboration
  • More time to innovate (rather than fix/maintain)

Disadvantages:

  • Requires a cultural change
  • Initial learning curve
  • Complexity with microservices
  • Security concerns
